只是简单的处理 csv 文件可以用 sql 进行处理, sqlite 提供快捷的操作.

```
# 在 csv 文件目录启动 sqlite3
sqlite3

# 设定分隔符
.separator ','

# 导入 csv 文件
.imprort t.csv <tablename>
```

csv 文件必须有表头信息
